# pdf_to_txt

The best choice for converting files from pdf to txt format

## Requirements

> python 3 and up

> module (Pypdf2, tqdm)

## install

```
git clone https://github.com/moayad-star/pdf_to_txt.git
```

```
cd pdf_to_txt 
```

## use


```python3 pdf_to_txt.py``` `file pdf`
